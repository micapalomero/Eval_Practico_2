Conceptos de Sistemas Informáticos - Fundamentos de Hardware.

  1. CPU (Unidad Central de Procesamiento):
    La CPU es el componente principal
    de un sistema informático encargado de ejecutar instrucciones y realizar
    operaciones aritméticas y lógicas. Es el "cerebro" del sistema y se comunica
    con otros dispositivos a través del bus del sistema.
  2. Memoria RAM (Memoria de Acceso Aleatorio):
    La RAM es una forma de memoria
    volátil que almacena temporalmente datos y programas en uso por el sistema
    operativo y las aplicaciones. Permite un acceso rápido a la información,
    pero pierde los datos cuando se apaga el sistema.
  3. Disco Duro:
    El disco duro es un dispositivo de almacenamiento de datos no
    volátil que almacena de forma permanente programas y archivos. Proporciona
    capacidad de almacenamiento a largo plazo y es crucial para conservar datos
    incluso cuando el sistema se apaga.
  4. Placa Base (Motherboard):
    La placa base es la plataforma principal que
    conecta y facilita la comunicación entre diferentes componentes del hardware,
    como la CPU, la memoria, las tarjetas de expansión y otros dispositivos periféricos.
    Contiene el BIOS/UEFI, que inicia el sistema.
  5. Sistema Operativo:
    El sistema operativo es un software fundamental que gestiona y
    coordina los recursos del hardware, permitiendo a los usuarios interactuar con la
    computadora. Proporciona una interfaz para ejecutar aplicaciones y controlar el
    hardware, gestionando tareas como el acceso a archivos, la memoria y la
    administración de procesos.
